### Apa itu HTML
#### HTML adalah HYpertext Markup Language

# Berikut contoh syntax HTML

```
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
</head>

<body></body>

</html>
```

# Sejarah HTML

HTML dimulai pada tahun 1991 ketika Tim Berners-Lee di CERN menciptakan versi pertama untuk berbagi dokumen penelitian, berkembang dari 18 tag awal menjadi standar web resmi yang dikelola W3C, dengan puncaknya adalah HTML5 (2014) yang membawa fitur semantik modern seperti <article> dan <footer>, memungkinkan konten web lebih kaya dan interaktif. 

# 1. Awal Penciptaan (1991â€“1993)

- 1991: Tim Berners-Lee merancang versi pertama HTML yang sangat sederhana, hanya mampu menampilkan teks, heading, paragraf, dan tautan (hyperlink).

- HTML 2.0 (1995): Memperkenalkan kemampuan formulir data, menjadi awal website yang lebih interaktif.

- HTML 3.0 & 3.2 (1997-1998): Menambahkan dukungan untuk tabel, rumus matematika, dan teks di sekitar gambar.

- TML 4.0/4.01 (1999): Menjadi standar resmi, mengadopsi CSS, mendukung skrip seperti JavaScript, dan memperkenalkan fitur aksesibilitas seperti alt tag.

- XHTML (2000): Versi yang lebih ketat, menggabungkan HTML dengan aturan XML (huruf kecil, tag wajib ditutup).

- HTML5 (2014): Versi paling mutakhir, membawa fitur semantik penting (<header>, <footer>) dan dukungan bawaan untuk audio/video, menjadi fondasi web modern.
