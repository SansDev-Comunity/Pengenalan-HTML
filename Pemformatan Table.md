# Pemformatan Table

Untuk memformat perataan text di dalam table Anda gunakan atribut Align dan Valign.

```
<table>
    <tr align="center" valign="baseline">
        <td>cell 1a</td>
        <td>cell 1b</td>
        <td>cell 1c</td>
    </tr>
    <tr align="center" valign="baseline">
        <td>cell 2a</td>
        <td>cell 2b</td>
        <td>cell 2c</td>
    </tr>
</table>
```
