# Komentar

Komentar sangat berguna untuk memberikan keterangan pada kode HTML.
Dengan adanya komentar maka kode akan lebih mudah di baca dan di mengerti
sehingga ketika ada kesalahan akan lebih mudah di perbaiki.

Contoh:
Buka Notepad++, kemudian tuliskan kode berikut dan simpan dalam file komentar.html

```
<html>

<head>
    <title>Komentar</title>
</head>

<body bgcolor = "blue"> <!- Warna BG jadi biru -->
    <h1>Frame di sebelah Kanan</h1>
</body>
</html>
```
Contoh di atas memiliki komentar dengan kode <!- warna bg jadi biru -->
