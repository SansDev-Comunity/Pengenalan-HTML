# Frame

Fungsionalitas bukanlah satu-satu nya faktor penting terhadap baik atau tidak sebuah web.
sebuah website mudah dan nyaman di gunakan, oleh karena itu website terdesain tampilan nya dengan baik.

Dalam HTML ada dua cara dasar layout sebuah website yaitu frame dan tag div.

Frame digunakan membagi jendela browser menjadi beberapa bagian terpisah.
paling sederhana daam me-layout sebuah halaman web, namun di HTML 5,
tag <frame> tidak lagi di gunakan.

Untuk membuat frame, tag di gunakan adalah:

- Tag FRAMESET
- Tag FRAME
- Tag NOFRAME
