# List

Untuk menampilkan daftar dalam halaman web, HTML memiliki tag khusus
jika daftar tersebut tidak terurut (Tampil menggunakan bullet),
maka tag yang di gunakan adalah <ul> yang berarti unorderlist </ul>
<ol> Ordered List </ol>

Contoh:
Buka Notepad++ tulis kode berikut

```
<html>

<head>
    <title>List</title>
</head>

<body>
  <h1> align="center">Tugas Akhir Kelompok TA 201</h1>
  <h1> align="center">Pengembangan SI Akademik ITB</h1>
  <ul>
    <li> Pendahuluan
      <ol>
        <li> Latar belakang
        <li> Rumusan Masalah
        <li> Tujuan
        <li> Batasan maslah
      </ol>
      <li> Daftar Pustaka
      <ol>
        <li Definisi Sistem Informasi
        <li> SI Akademik ITB saat ini
      </ol>
  </ul>
</body>
</html>
```
