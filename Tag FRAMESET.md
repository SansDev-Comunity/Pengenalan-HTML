# Tag FRAMESET

Frameset mempunyai beberapa atribut

| Atribut | Keterangan 
| :--- | :--- | 
| ROWS | Membuat frame secara horizontal segaligus mendefinisikan lebar masing-masing
| COLS | Membuat Frame secara vertikal sekaligus mendefinisikan lebar masing-masing
| BORDERCOLOR | menentukan warna bingkai frame
