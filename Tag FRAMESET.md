# Tag FRAMESET

Frameset mempunyai beberapa atribut

| Atribut | Keterangan 
| :--- | :--- | 
| ROWS | Membuat frame secara horizontal segaligus mendefinisikan lebar masing-masing
| COLS | Membuat Frame secara vertikal sekaligus mendefinisikan lebar masing-masing
| BORDERCOLOR | menentukan warna bingkai frame

Untuk mendefinisikan  lebar frame pada atribut ROWS dan COLS ada tiga cara dapat di lakukan

Dengan nilai tetap
FRAMESET ROWS = "50,140,400,50"

Dengan nilai presentase
FRAMESET COLS = "20%, 40%, 20%, 20%"

Dengan nilai proporsional
FRAMESET ROWS = " *,2* , *,* "
