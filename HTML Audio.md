# HTML Audio

```
<!DOCTYPE html>
<html>
<head>
	<title>Audio</title>
</head>
<body>
	<h1>Sample Audio</h1>
		<audio controls autoplay>
 			<source src="audio/viper.mp3" type="audio/mp3"> 
  			<p>Your browser doesn't support HTML5 audio. Here is a <a href="audio/viper.mp3">link to the audio</a> instead.</p>
		</audio>
</body>
</html>
```
