# Tag dan Elemen HTML

penanda perintah di HTML di sebut Tag.
untuk membuat tampilan di dokumen yang berpasangan dengan format berikut:

```
<BEGIN TAG> isi dokumen </END TAG>
```

```
<TAG isi dokumen
```

Perlu di perhatikan bahwa tag

```
<p>
```
lebih baik di akhiri dengan tag

```
</p>
```

Contoh: setiap dokumen HTML di awali dan diakhiri dengan HTML

```
<html>

  <head>
  ...
  </head>

  <body>
  ...
  </body>

</html>
```
