# Link

Link menjadi penghubung antara satu halaman dengan halaman lain,
dengan satu alamat tertentu.
jika link di klik maka dokumen HTML akan menuju ke halaman anchor, yaitu tag 
```
a
```

Contoh:
buka Notepad++, kemudian tuliskan kode berikut

```
<HTML>

<HEAD>
    <TITLE> Penggunaan Link </TITLE>
</HEAD>

<BODY>
    <A HREF="https://san-digital-agency.vercel.app/"> klik disini </A> untuk menuju situs san-digital-agency.vercel.app
    <BR> atau dapat juga mengklik gambar ini :
    <A HREF="https://san-digital-agency.vercel.app/">
        <IMG SRC="https://raw.githubusercontent.com/FebrianyRenata02/San-Digital-Academy/refs/heads/main/img/Agency%20Logo.png" ALT="educativegames.net" width="300px">
    </A>
    <BR>
    <BR>
</BODY>


</HTML>
```
